# Stock Market Trend Prediction using Sentiment Analysis

## Overview
This project predicts short-term stock market trends (UP / DOWN / NEUTRAL)
for selected NIFTY 50 companies by combining:
- News sentiment analysis using NLP (VADER)
- Recent stock price momentum using Yahoo Finance data

The notebook is structured with clear problem definition, data understanding,
system design, evaluation, and ethical considerations.

## How to Run
1. Open the notebook in Google Colab or Jupyter Notebook
2. Run all cells from top to bottom
